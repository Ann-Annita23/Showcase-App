## Project Showcase App
A clean, responsive React application designed to help users organize, search, and manage their creative projects. This application uses a modular component structure and is styled with Bootstrap for a modern, professional look.

## 🚀 Features
1. Add Projects: Quickly add new projects with a title and description via a structured form.

2. Real-time Search: Filter through your project list instantly using the search bar.

3. Responsive Design: A two-column layout that shifts seamlessly from desktop to mobile views.

4. State Management: Utilizes React useState to handle dynamic list updates and search filtering.

## 🛠️ Built With
1. React: For building the user interface and managing component logic.

2. Bootstrap 5: For responsive layout, cards, and form styling.

3. CSS Modules: For specific component-level styling where needed.

## 📂 Project Structure
1. Header.jsx: Displays the application title and branding.

2. ProjectForm.jsx: The input area for adding new project titles and descriptions.

3. SearchBar.jsx: The interface for filtering the displayed projects.

4. ProjectList.jsx: The engine that filters and renders individual project cards.

## How  to set it up 
1. Clone the repository
In gitHub git clone https://github.com/Ann-Annita23/Showcase-App#

2. Install dependencies
npm install
npm install bootstrap

3. Run the application
npm run dev

## 📝 Usage
Adding a Project: Enter a title and description in the "Add Project" sidebar and click the button to save it to your list.

Searching: Type any part of a project title into the search input to narrow down your view.
